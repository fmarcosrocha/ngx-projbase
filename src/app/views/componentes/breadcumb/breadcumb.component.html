Componente customizado

Por se tratar de um componente que não é fornecido nativamente pelo angular-material, foi construído e customizado o componente a seguir.

Breadcumb

Breadcrumb, Breadcrumb Navigation ou Breadcrumb Trail (literalmente, navegação por migalhas de pão ou trilha de migalhas de pão, em alusão à história de João e Maria) é um sistema de navegação estrutural usado para proporcionar ao usuário um meio de localização dentro da estrutura de navegação do site ou sistema.

 
<mj-breadcumb></mj-breadcumb>

O componente irá se montar de acordo com os dados da rota, para que o mesmo funciona será necessário ajustar o seu route para o seguinte:

Atente-se a propriedade data.

...
path: 'componentes', data: {{ '{' }} breadcrumb: 'Componentes', rota: "/componentes" {{ '}' }}, children: [
  {{ '{' }} path: 'breadcumb', component: BreadcumbComponent, data: {{ '{' }} breadcrumb: 'Breadcumb', rota: "/componentes/breadcumb" {{ '}' }} {{ '}' }},
...
