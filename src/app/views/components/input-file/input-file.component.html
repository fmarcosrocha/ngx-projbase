custom component

As this is a component that is not natively provided by angular-material, the following component was built and customized.

A directive was also built to allow dragging and dropping files.

File upload

It is the component that allows the input of textual data by the user.

Upload Example
cancel{{ uploadCtrl.error }} 
<mat-label>Upload Example</mat-label>
<div class="mat-button-upload">
  <input hidden type="file" #inputFileUpload (change)="uploadCtrl.upload($event)"/>
  <button class="w-100" mat-stroked-button color="primary" (click)="inputFileUpload.click()" libFileDrop (files)="uploadCtrl.dropped($event)">
    <div class="d-flex">
      <mat-icon class="mr-3">upload</mat-icon>
      <span>Select a file</span>
    </div>
  </button>
  <div *ngIf="uploadCtrl.error" class="mat-error">
    <mat-icon aria-hidden="false">cancel</mat-icon>{{'{{'}} uploadCtrl.error {{'}}'}}
  </div>
</div>

public uploadCtrl = {{'{'}}
  error: '',
  upload: (evt: any) => {{'{'}}
    this.uploadCtrl.error = 'Files must be in PNG or JPG format and have a maximum size of 100MB';
  {{'}'}},
  dropped: (files: any) => {{'{'}}
    this.uploadCtrl.error = 'Files must be in PNG or JPG format and have a maximum size of 100MB';
  {{'}'}}
{{'}'}}
