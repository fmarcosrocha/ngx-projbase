Componente customizado

Por se tratar de um componente que não é fornecido nativamente pelo angular-material, foi construído e customizado o componente a seguir.

Também foi construído uma diretiva para possibilitar o arrastar e soltar arquivos.

File upload

É o componente que permite a entrada de dados textuais por parte do usuário.

Exemplo de upload
cancel{{ uploadCtrl.error }} 
<mat-label>Exemplo de upload</mat-label>
<div class="mat-button-upload">
  <input hidden type="file" #inputFileUpload (change)="uploadCtrl.upload($event)"/>
  <button class="w-100" mat-stroked-button color="primary" (click)="inputFileUpload.click()" libFileDrop (files)="uploadCtrl.dropped($event)">
    <div class="d-flex">
      <mat-icon class="mr-3">upload</mat-icon>
      <span>Selecione o arquivo</span>
    </div>
  </button>
  <div *ngIf="uploadCtrl.error" class="mat-error">
    <mat-icon aria-hidden="false">cancel</mat-icon>{{'{{'}} uploadCtrl.error {{'}}'}}
  </div>
</div>

public uploadCtrl = {{'{'}}
  error: '',
  upload: (evt: any) => {{'{'}}
    this.uploadCtrl.error = 'Os arquivos devem ser no formato PNG ou JPG e ter no máximo 100MB';
  {{'}'}},
  dropped: (files: any) => {{'{'}}
    this.uploadCtrl.error = 'Os arquivos devem ser no formato PNG ou JPG e ter no máximo 100MB';
  {{'}'}}
{{'}'}}
