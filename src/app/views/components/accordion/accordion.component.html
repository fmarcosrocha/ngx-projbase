Accordion

The accordion is used so that the user can see more information about a content, as it allows the initially displayed item on the screen in the form of a sentence to be expanded, showing the details of that item.

Its use is indicated in extensive contents, especially in mobile device interfaces, in order to reduce the density of information for the user.
