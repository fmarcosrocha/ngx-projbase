custom component

As this is a component that is not natively provided by angular-material, the following component was built and customized.

Breadcumb

Breadcrumb, Breadcrumb Navigation or Breadcrumb Trail (literally breadcrumb navigation or breadcrumb trail) breadcrumbs, alluding to the story of Hansel and Gretel) is a structural navigation system used to provide the user a means of location within the navigational structure of the site or system. 

<lib-breadcumb></lib-breadcumb>

The component will assemble according to the route data, for it to work you will need to adjust your route to the following:

Pay attention to the property data.

...
path: 'components', data: {{ '{' }} breadcrumb: 'components', route: "/components" {{ '}' }}, children: [
  {{ '{' }} path: 'breadcumb', component: BreadcumbComponent, data: {{ '{' }} breadcrumb: 'Breadcumb', route: "/components/breadcumb" {{ '}' }} {{ '}' }}
  ,
...
