Como começar

A seguir, será explicado como subir e desenvolver neste projeto.

O projeto é composto por Componentes de exemplo, para ajudar outros times a consumirem e por uma biblioteca que irá fornecer o layout base e um tema/sass (com classes customizadas) para o Angular-Material


Como os projetos consomem nossa lib?

Este projeto irá gerar um arquivo compilado, mjbase-*.tgz, e esse será importado nos projetos cliente. Mais informações na documentação de instalação.


Estrutura do projeto

  • Componentes de exemplo (O que for alterado aqui, não afeta os projetos clientes desta arquitetura)
    • src/app
  • Biblioteca base (O que for alterado aqui afeta os projetos clientes, desde que eles consumam uma nova versão do build)
    • projects/projbase

Instalação das dependências

npm install

Build e pack da biblioteca base

No package.json foi configurado os seguintes comandos:

npm run build:library
npm run pack:library

Rodando o projeto

npm start
