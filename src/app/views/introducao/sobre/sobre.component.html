O projeto

O projeto Arquitetura Identidade Visual - Angular - Core Applications consite em uma adaptação visual do Angular-Material para aplicar a identidade visual do que foi estabelecido nos projetos:

  • gov.br - 2.0.2 BETA (https://dsgov.estaleiro.serpro.gov.br/)
  • oscip - 1.0.0 (https://staging-aplicacao.mj.gov.br/oscip-dsv/)

Aqui você vai obter a folha de estilo (SASS) e os componentes base do projeto, que são:

  • Menu superior
  • Menu lateral
  • Rodapé

Arquitetura base

A arquitetura base foi construída considerando o Angular.io v11 e o Angular-Material v11, seguindo as orientações de customização de tema para o Angular-Material.


Componentes base:

@angular/core: ~11.2.11
@angular/material: ^11.2.13

O que você precisa?

Se você vai trabalhar neste projeto base, fazendo alguma customização ou correção, siga aqui.
