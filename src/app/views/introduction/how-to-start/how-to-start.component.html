How to get started

The following will explain how to go up and develop in this project.

The project is made up of Example components, to help other teams consume them, and a library that will provide the base layout and a theme/sass (with classes custom) for Angular-Material


How do projects consume our lib?

This project will generate a compiled file, projbase-*.tgz, and this will be imported into client projects. More information in the installation documentation.


Project Structure

  • Sample Components (What you change here does not affect client projects of this architecture)
    • src/app
  • Base library (What you change here affects client projects as long as they consume a new build version)
    • projects/projbase

Installing dependencies

npm install

Build and pack the base library

The following commands were configured in package.json:

npm run build:library
npm runpack:library

Running the project

npm start
